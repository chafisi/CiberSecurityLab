Caso Práctico 1

Verificación de Primalidad (Miller-Rabin)

Enunciado: ¿Es 104729 primo? Describe el proceso incluyendo el test de Miller-Rabin.
Solución Teórica
  1. Métodos básicos iniciales: Probar división por primos pequeños hasta √n ≈ 323
  2. Test de Miller-Rabin: Expresar n−1 = d⋅2ˢ (ej. 104728 = 13091 ⋅ 2³)
  3. Elegir base aleatoria a (ej. a=2) y calcular x = aᵈ(mod n)
  4. Si x=1 o x=n−1 → probable primo. Repetir s−1 veces x=x²(mod n)
  5. Resultado: Al ejecutar con múltiples bases, cumple condiciones → ES PRIMO

Inverso Multiplicativo en RSA (Euclides Extendido)

Enunciado: ¿Cómo encontrar el inverso multiplicativo de 'e' módulo 'phi'?

Cifrado Simétrico (DES)

Enunciado: Proceso de cifrado/descifrado con DES y gestión de clave.
Solución Teórica
  1. Gestión de Clave: Alice y Bob comparten previamente clave secreta única (56 bits efectiva)
  2. Cifrado (Alice): Divide mensaje en bloques de 64 bits, aplica DES con clave compartida
  3. Transmisión: Envía texto cifrado por canal inseguro
  4. Descifrado (Bob): Recibe bloque y usa misma clave secreta para descifrar

Caso Práctico 4

Cifrado Asimétrico (RSA) - Confidencialidad

Enunciado: Cómo Alice cifra con la clave pública de Bob y Bob descifra.
Solución Teórica
  1. Generación de claves (Bob): Pública (n,e) y Privada (n,d). Comparte Pública
  2. Cifrado (Alice): Convierte mensaje M en número, calcula C = Mᵉ(mod n)
  3. Transmisión: Envía C a Bob. Incluso interceptando C, n, e no se recupera M
  4. Descifrado (Bob): Usa clave privada d: M = Cᵈ(mod n)

Caso Práctico 5

Integridad con SHA-256

Enunciado: Cómo Alice garantiza integridad usando SHA-256.
Solución Teórica
  1. Generación de Hash: Alice calcula H(M) = SHA-256(M)
  2. Envío: Envía mensaje M junto con hash H(M)
  3. Verificación (Bob): Calcula su propio hash H(M') = SHA-256(M')
  4. Comparación: Si H(M') == Hrecibido → Íntegro. Si ≠ → Modificado

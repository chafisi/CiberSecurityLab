🔐 BLOQUE I: CRIPTOGRAFÍA

Casos Prácticos 1-6 | Seguridad Simétrica y Asimétrica

Caso Tema Método/Símbolo Flujo Clave (Siglas)
#1 Primalidad MR Test $n-1 = d·2^s$ → Base a → $x=a^d \bmod n$ → Iterar $x^2$ s veces
#2 Factorización Euclides Exp. $\text{mcd}(a,b)$ para encontrar inverso modular d en RSA
#3 Simétrico DES Clave 56bits → Bloque 64bit → Feistel Network → Mismo key cifrar/descifrar
#4 Asimétrico RSA Generar $(p,q,n,\phi,e,d)$ → Cifrar $C=M^e \bmod n$ → Descifrar $M=C^d \bmod n$
#5 Simétrico Moderno AES-CBC Clave 128/256bit → IV aleatorio → Bloque XOR + Cifrado → Encadenado
#6 Integridad SHA-256 Calcula Hash(H) → Envía [Msg+H] → Recibe calcula H' → Compara H=H'? ✅

🛡️ Regla Rápida Cripto

SIMÉTRICO

AES/DES → 1 clave rápida, distribución difícil

ASIMÉTRICO

RSA/ECC → 2 claves segura, intercambio sin canal seguro

HASH

SHA-256 → Integridad, no reversible, efecto avalancha

🔗 BLOQUE II: BLOCKCHAIN

Casos Prácticos 7-14 | Consenso y Smart Contracts

Caso Tema Método/Símbolo Flujo Clave (Siglas)
#7 Estructura Bloque Block Header Tx Data + HashAnt + Timestamp + Nonce + HashActual
#8 Minado PoW Proof of Work Agrupar Tx → Buscar nonce con Hash < Target → Verificación red → Recompensa
#9 Minado PoS Proof of Stake Apuesta (Staking) → Selección aleatoria → Propone bloque → Attestation → Slashing si mal

⛓️ Regla Rápida Blockchain

POW

(Bitcoin) Energía, mineros resuelven puzzles, seguro pero lento

POS

(ETH 2.0) Apuesta monedas, validadores, eficiente energía

SMART CONTRATO

If-then auto, immutabilidad, código = ley

1. Números Primos y Divisibilidad

🔢 El Pilar de la Criptografía

Los números primos son los "átomos" de las matemáticas. En criptografía asimétrica (como RSA), la seguridad radica en la facilidad de multiplicar dos primos enormes para obtener un módulo n, pero la extrema dificultad computacional de hacer lo inverso (factorizar n).

➗ Aritmética Modular

Es la "matemática del reloj". En lugar de crecer infinitamente, los números se reinician al alcanzar un valor máximo (el módulo).

Ejemplo: Si son las 10:00 y pasan 5 horas, no son las 15:00 en un reloj de 12 horas, son las 3:00. Matemáticamente: (10 + 5) mod 12 = 3.

2. Algoritmos Fundamentales

El Algoritmo de Euclides Extendido

El algoritmo clásico encuentra el Máximo Común Divisor (MCD). La versión extendida hace un seguimiento de los coeficientes para expresar el MCD como una combinación lineal. En RSA, esto es vital para calcular la Clave Privada d.

Objetivo: Encontrar 'd' tal que (e * d) ≡ 1 mod φ(n)

1. Planteamos la ecuación: e*x + φ(n)*y = MCD(e, φ(n)) = 1

2. Aplicamos divisiones sucesivas hacia atrás.

3. El coeficiente 'x' obtenido será nuestra 'd'. Si es negativo, le sumamos φ(n).

Test de Miller-Rabin

Es un test probabilístico. No prueba que un número es primo con 100% de certeza, pero ejecutado varias veces, reduce la probabilidad de que sea compuesto a un margen matemáticamente despreciable. Es rápido e ideal para generar primos de 2048 bits.

Coprimalidad

Dos números son coprimos si su MCD es 1 (no comparten factores aparte del 1). En RSA, el exponente público e DEBE ser coprimo con φ(n) para que exista su inverso multiplicativo.

4. Fichas Rápidas (Memorización)

➗

Algoritmo de Euclides

MCD

Método eficiente para calcular el Máximo Común Divisor de dos números. Se basa en el principio: mcd(a, b) = mcd(b, a mod b).

🔄

Inverso Multiplicativo

Módulo

El inverso de 'e' módulo 'n' es el número 'd' tal que (e * d) mod n = 1. Es la base para calcular la clave privada en RSA.

🧮

Función φ de Euler

Totiente

Cuenta los enteros positivos menores a 'n' que son coprimos con 'n'. Si n = p*q (primos), entonces φ(n) = (p-1)*(q-1).

🎲

Miller-Rabin

Probabilidad

Algoritmo que determina si un número es compuesto o "probablemente primo". Fundamental para no eternizarse factorizando números de 2048 bits.

Caso Práctico 1: Verificación de Primalidad

Solución

Pregunta: ¿Es 104729 primo? Describe el proceso incluyendo el test de Miller-Rabin.

Solución Técnica:

Para determinar si n = 104729 es primo:

  • Métodos básicos iniciales: Se prueba división por primos pequeños (2, 3, 5...) hasta √n ≈ 323. Si no hay divisores, se pasa a tests probabilísticos.
  • Test de Miller-Rabin:
    • Expresamos n-1 = 104728 como d · 2s.
    • 104728 = 13091 · 23 (donde d = 13091 es impar y s = 3).
    • Elegimos una base a aleatoria tal que 1 < a < n-1 (ej. a = 2).
    • Calculamos x = ad (mod n) = 213091 (mod 104729).
    • Si x = 1 o x = n-1, entonces n es probable primo para esta base.
    • Si no, repetimos s-1 veces: x = x2 (mod n). Si alguna vez x = n-1, es probable primo. Si al final no llegamos a n-1, es compuesto.
  • Resultado: Al ejecutar el algoritmo con múltiples bases (ej. 2, 3, 5), 104729 cumple las condiciones para todas.
  • Conclusión: Sí, 104729 es un número primo. El test confirma la primalidad con alta probabilidad (cercana a 1).

ESQUEMA 1: PRIMALIDAD (MILLER-RABIN)

  • 1️⃣ BASE 🔢 n=104729. Se descartan primos hasta √n.
  • 2️⃣ DESCOMPOSICIÓN ⚙️ n-1 = d * 2^s -> 104728 = 13091 * 2^3.
  • 3️⃣ ITERACIÓN 🔄 Elegir base 'a'. Evaluar x = a^d mod n. Cuadrar si es necesario.
  • 4️⃣ RESULTADO ✅ Cumple para bases 2, 3, 5. Es PRIMO.

🛡️ Fundamental para generar claves seguras sin colapso computacional.

Caso Práctico 2: Inverso Multiplicativo en RSA

Solución

Pregunta: ¿Cómo encontrar el inverso multiplicativo de 'e' módulo 'phi'?

Solución Técnica:

  • Concepto: En RSA, la clave privada d es el inverso modular de e módulo φ(n). Es decir, buscamos d tal que e · d ≡ 1 (mod φ(n)).
  • Datos Iniciales: Tenemos e y φ(n) (calculado como (p-1)(q-1)). Deben ser coprimos: mcd(e, φ(n)) = 1.
  • Algoritmo de Euclides Extendido: Este algoritmo encuentra enteros x e y tales que: e · x + φ(n) · y = mcd(e, φ(n)) = 1.
  • Proceso Matemático:
    • Se realizan divisiones euclidianas sucesivas hasta llegar al residuo 1.
    • Se sustituyen hacia atrás (expansión) para expresar 1 como combinación lineal de e y φ(n).
    • El coeficiente de e es el valor x.
  • Ajuste y Resultado: Si x es negativo, se suma φ(n) hasta obtener un valor positivo (d = x (mod φ(n))). Ese valor d es la clave privada utilizada para descifrar.

ESQUEMA 2: EUCLIDES EXTENDIDO (RSA)

  • 1️⃣ OBJETIVO 🎯 Hallar 'd' tal que e * d ≡ 1 mod φ(n).
  • 2️⃣ REQUISITO ⚖️ e y φ(n) deben ser coprimos (MCD = 1).
  • 3️⃣ EXPANSIÓN 🧮 e*x + φ(n)*y = 1. Sustitución hacia atrás.
  • 4️⃣ CLAVE PRIVADA 🗝️ d = x. Si es negativo, sumar φ(n).

🛡️ Este cálculo es la piedra angular para crear la clave secreta de descifrado en RSA.

6. Glosario Técnico y Matemático

Congruencia (Aritmética Modular)

Dos números a y b son congruentes módulo n (a ≡ b mod n) si ambos tienen el mismo residuo al ser divididos por n.

Factorización de Enteros

El proceso de descomponer un número compuesto en sus divisores primos. Su extrema dificultad computacional para números grandes asegura el sistema RSA.

Test Probabilístico

Algoritmo que proporciona una respuesta que es correcta con una probabilidad muy alta, pero no garantiza certeza absoluta. Ej. Miller-Rabin.

Coprimos

Dos números son coprimos o primos relativos si no tienen ningún divisor primo en común. Su Máximo Común Divisor (MCD) es estrictamente 1.

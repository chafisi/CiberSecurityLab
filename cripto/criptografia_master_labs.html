ESTRUCTURA OFICIAL RSA

Clave Pública

(e, n)

Se comparte. Sirve para cifrar el mensaje.

Ecuación de Cifrado:

C = Me mod n

Clave Privada

(d, n)

Se oculta. Sirve para descifrar el criptograma.

Ecuación de Descifrado:

M = Cd mod n

Esquema Mental Acelerado: Requisitos, Flujos y Validación

📦 REQ: [F] Aislar/Backups [NF] RTO<4h / Disp. [T] EDR / Segmentación L2/L3 [L] RGPD

✅ VAL: Simulacro Phishing, Test Restore Backups.

📦 REQ: [F] MFA/Bloqueo [NF] Auth <2s [T] Argon2/Bcrypt / SIEM [L] PSD2/RGPD
🔄 Flujo: Usuario ➔ Loguea ➔ Solicita MFA ➔ Accede ➔ (Exc: CAPTCHA/Bloqueo temporal)
✅ VAL: Hydra/Burp (Fuerza bruta), Test Funcional MFA.

3. Transición DES a AES

📦 REQ: [T] OpenSSL/AES-256 [NF] Compatibilidad
🔄 Flujo: Admin ➔ Detecta obsoleto ➔ Migra AES ➔ (Exc: Mitiga legacy)

13. Certs. Caducados

📦 REQ: [T] WebAuthn / Hardware Key [F] Anti-Phishing

5 & 6. Firewall & NGFW

📦 REQ: [F] Reglas previas / DPI [T] Staging / App-ID / SIEM
🔄 Flujo: Admin ➔ Prueba Staging ➔ NGFW DPI ➔ Bloquea Tor/VPN ➔ (Exc: Falso Positivo)
✅ VAL: Nmap (Escaneo), Config Drift.

7 & 8. IDS (Snort) / IPS

📦 REQ: [NF] Baja tasa FP / Disp. 99% [T] SPAN/TAP / Firmas
🔄 Flujo: Snort ➔ Detecta SYN Scan ➔ IPS Bloquea ➔ (Exc: Tráfico Legítimo)
✅ VAL: Traffic Replay, Tuning de reglas.

9 & 10. ARP (L2) & IPSec (L3)

📦 REQ: [T] DAI / IKEv2 / AES-GCM / PFS
🔄 Flujo 9: Switch ➔ DAI valida MAC ➔ Drop.
🔄 Flujo 10: Sedes ➔ Negocian IKEv2 ➔ Túnel ESP.
✅ VAL: Ettercap (Test ARP), Wireshark (Test Túnel).
🌐 BLOQUE D: AppSec y Transmisión de Información (L7)

11/12. Patches & XSS

📦 REQ: [F] Sanear Input / Flujo CI-CD [T] WAF / CSP / Cookies HttpOnly
🔄 Flujo: Servidor ➔ Sanitiza input ➔ Codifica salida ➔ (Exc: Payload bloqueado)
✅ VAL: BurpSuite (DAST XSS), Nessus (SCA).

15/16. Huffman & CRIME

📦 REQ: [T] TLS 1.3 / Padding Random / No HPACK co-compress
🔄 Flujo: App ➔ Aísla secretos ➔ Comprime ➔ Cifra ➔ (Exc: Tamaño revela info -> Padding)
✅ VAL: Oráculos de longitud (Variación Ciphertext).

17/18. Reed-Solomon & LDPC

📦 REQ: [F] Corregir BER [T] Parámetros RS / Belief Propagation 5G
🔄 Flujo: Receptor ➔ Sube Ruido ➔ Itera LDPC/RS ➔ (Exc: Jamming extremo -> Handover)
✅ VAL: Generador Ruido (Test SNR/Jamming).
⚖️ BLOQUE E: GRC, Legal y Marco Normativo
